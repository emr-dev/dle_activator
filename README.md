# DLE Activator
Script for automatic activation of the DLE engine.

## Version support

Works on DLE 14 version.
Should work on version 13+ too.

## Using 

1. Upload the **dlekey.php** file to the root of the project directory

3. Run and enjoy

  ```
  <host>/dlekey.php
  ```
